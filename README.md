## Run locally
```bash
npm install
npm run dev
```

## Deploy (auto)
Push to `main` → GitHub Actions builds and publishes to `gh-pages` automatically.
Then enable **Settings → Pages → Source: gh-pages (root)**.

Live at:
```
https://<your-username>.github.io/VirtualWarfareWebsite/
```
